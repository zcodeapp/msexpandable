# Interfaces

## Overview

This module provides a interfaces.

## Installation

Include the `Interfaces` class in your TypeScript project:

```bash
npm install @zcodeapp/interfaces
```

## Usage

```typescript
import { IExample } from "@zcodeapp/interfaces";

const example: IExample;
```
